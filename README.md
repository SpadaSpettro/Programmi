This repository contains very small C++ programs that perform simple mathematical operations:

* Program: prompts you to enter an integer and, depending on the number entered, performs different operations.
* Program 2: calculates the sum and average of 10 numbers, plus a special calculation!



The repository also contains a calculator program in Python.



###### How to run the program:

* To run a C++ program, compile its .cpp file using a C++ compiler (such as g++ or Visual Studio) and run the generated executable.
* To run the Python program, compile the .py file using a Python compiler.
